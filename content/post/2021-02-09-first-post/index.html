Welcome to my site!

As a newbie to the world of website architecture, I figured it was appropriate to begin this blog with a common coding sanity-check.

Hello World!

Ok, I think it’s working.

Purpose

I’m hoping this site will provide:

  • A platform to share and document things I’m working on and learning about, for my future self and for anyone else who happens to stumble upon it.
  • A means for past, present, and future collaborators and employers to learn more about me and my work.
  • A way to practice communicating my professional work and fun side projects.
  • An excuse to experiment and play with static website development.

How I created this site

I created this site using the R package blogdown, which uses Hugo and R Markdown to build websites. The blogdown documentation has proved quite useful and is actively being updated. Additionally, I’ve found the Wowchemy Documentation and Markdown syntax guide to be handy resources. Finally, I’ve learned quite a bit by simply experimenting.
